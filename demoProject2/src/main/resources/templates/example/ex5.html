안전 탐색 연산자: ${객체?.필드}

        - 객체가 null인지 판단 후 null이 아닌 경우에 수행

${#number.sequence}(시작, 끝, [, step]) : 번호 생성기

        - 시작부터 끝 까지 번호로만 이루어진 숫자 배열 생성

        ex) ${#number.sequence}(1, 5); => {1,2,3,4,5}

        [ 왜 사용할까? ]
        - 타임리프 반복문 th:each는 향상된 for문으로
          일반 for문의 동작(시작부터 끝까지 몇 씩 증가하면서 반복)을 못하기 때문에
          이를 해결하기 위해서 사용
